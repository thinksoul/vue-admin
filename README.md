# vue-admin
## 项目说明
后台管理系统模板， 加入动态路由，权限验证,根据权限显示相关侧边栏。
使用 json-server 模拟mock数据。
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
